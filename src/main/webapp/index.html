Todos Application

This is a sample Todos application built with Jakarta EE 10 and Payara 6. It demonstrates a RESTful API for managing users, todo lists, and tasks.

Available REST Endpoints

  • /todos/api/users - Manage users
  • /todos/api/todolists - Manage todo lists
  • /todos/api/tasks - Manage tasks

How to Run

Follow these steps to run the application locally:

  1. Create the JDBC connection pool and resource in Payara as described in the README.
  2. Build and start the application using the Maven wrapper:
chmod +x mvnw
./mvnw clean package payara-server:dev

Access the API

Once running, you can access the API at http://localhost:8080/todos/api/. For example, to get all tasks, visit /todos/api/tasks.

About

This project is generated by the Payara Starter and demonstrates Jakarta EE 10 features with Payara 6. It uses H2 database for persistence and Flyway for database migrations.

For more information, visit the Payara website and documentation.
